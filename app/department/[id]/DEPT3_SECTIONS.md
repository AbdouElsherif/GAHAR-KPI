# دليل أقسام الإدارة العامة لرضاء المتعاملين (dept3)

يحتوي ملف `app/department/[id]/page.tsx` على الأقسام التالية الخاصة بالإدارة العامة لرضاء المتعاملين. يمكن البحث باستخدام العلامات المرجعية المذكورة أدناه للتنقل السريع.

| القسم | العلامة المرجعية (Search Marker) | الوصف |
| :--- | :--- | :--- |
| **المؤشرات الرئيسية** | `DEPT3-SECTION-1` | النموذج والجدول العام الذي يعرض ملخص البيانات والمؤشرات الأساسية لرضاء المتعاملين. |
| **استبيانات رضاء المتعاملين حسب المحافظة** | `DEPT3-SECTION-2` | عرض تفاصيل الاستبيانات التي تم إجراؤها في مختلف المحافظات (استخدام المكون `GovernorateCustomerSurveysSection`). |
| **لوحة البيانات (Dashboard)** | `DEPT3-SECTION-3` | المكون البرمجي `CustomerSatisfactionDashboard` الذي يعرض التحليلات والرسوم البيانية داخل نافذة منبثقة. |

## كيفية الاستخدام
1. افتح ملف `app/department/[id]/page.tsx`.
2. اضغط على `Ctrl + F` للبحث.
3. اكتب اسم العلامة (مثلاً: `DEPT3-SECTION-2`) للذهاب مباشرة إلى الكود الخاص بهذا القسم.
