# دليل أقسام الإدارة العامة للرقابة الإدارية (dept5)

يحتوي ملف `app/department/[id]/page.tsx` على الأقسام التالية الخاصة بالإدارة العامة للرقابة الإدارية على المنشآت الصحية. يمكن البحث باستخدام العلامات المرجعية المذكورة أدناه للتنقل السريع.

| القسم | العلامة المرجعية (Search Marker) | الوصف |
| :--- | :--- | :--- |
| **المؤشرات الرئيسية** | `DEPT5-SECTION-1` | الجدول العام الذي يعرض ملخص البيانات والمؤشرات الأساسية. |
| **المنشآت التي تمت زيارتها** | `DEPT5-SECTION-2` | عرض تفاصيل المنشآت التي شملتها جولات الرقابة الإدارية خلال الشهر. |
| **الملاحظات المتكررة** | `DEPT5-SECTION-3` | قائمة بالملاحظات الأكثر شيوعاً التي تم رصدها خلال الزيارات. |
| **نسب تصحيح الملاحظات** | `DEPT5-SECTION-4` | قياس مدى استجابة المنشآت وتصحيحها للملاحظات المرصودة. |
| **لوحة البيانات (Dashboard)** | `DEPT5-SECTION-5` | المكون البرمجي `AdminAuditDashboard` الذي يعرض التحليلات والرسوم البيانية. |

## كيفية الاستخدام
1. افتح ملف `app/department/[id]/page.tsx`.
2. اضغط على `Ctrl + F` للبحث.
3. اكتب اسم العلامة (مثلاً: `DEPT5-SECTION-2`) للذهاب مباشرة إلى الكود الخاص بهذا القسم.
